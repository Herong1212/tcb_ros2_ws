# TCB 系列机械臂 SDK
## 概述

本 SDK 适用于 **TCB 系列机械臂**，基于 TCB 机械臂控制器上位机通讯协议 **V22.07** 实现，用于上位机与机械臂控制器之间的通信与运动控制。

SDK 提供机械臂的基础控制接口、运动控制接口以及基于 SDK 的二次开发示例，可用于算法验证、系统集成及应用开发。

## 版本说明
- 适配机械臂控制器通讯协议版本：**V22.07**
- 适用于 **TCB 系列 6 / 7 自由度机械臂**

## 使用前准备
### 1. 机械臂自由度配置

在使用 SDK 前，请根据 实际机械臂的自由度 修改以下配置文件：

`./config/tcb_cfg.yaml`

需要重点确认并修改参数：

`DOF: <机械臂自由度>`

如果使用运动学相关功能，根据机械臂类型修改参数：

`ARM_TYPE: <机械臂类型>`

目前支持四个型号机械臂：TCB605_05、TCB610_06、TCB705_05、TCB710_06

### 2. 网络连接配置

请确保上位机与机械臂控制器处于同一网络环境，上位机能够 ping 通机械臂控制器 IP

机械臂控制器对外开放以下端口：

| 端口号 | 功能说明 |
|------|--------|
| 6001 | 通用控制与状态查询 |
| 7000 | 运动控制 / 伺服点位运动控制 |

## 通讯端口说明
### 1. 6001 端口（通用功能端口）

6001 端口主要用于机械臂的通用控制功能，包括但不限于：

- 机械臂初始化

- 状态查询

- 参数设置

- 错误清除等

具体接口定义与使用方式请参考：

`robotarm_sdk.py`

### 2. 7000 端口（运动控制端口）

7000 端口主要用于机械臂的运动控制，包含以下两类功能：

- 运动控制接口（推荐使用）

- 伺服点位运动控制

具体接口定义与使用方式请参考：

`robotarm_sdk.py`

## 目录结构说明

```text
.
├── config
├── example
├── log
├── models
├── utils
```

### 1. ./config

配置文件目录，包含 SDK 运行所需的核心配置文件：

 - `constants.yaml`
   - 各功能在通讯协议 V22.07 中对应的命令字
 
 - `tcb_cfg.yaml`
   - 机械臂基础配置文件（如自由度等）

### 2. ./example

示例代码目录，包含基于 SDK 的基础使用示例，用于快速上手与功能验证。

### 3. ./log

日志目录，用于存储 SDK 运行过程中产生的日志信息，便于调试与问题定位。

### 4. ./models

机械臂模型文件目录，包含 TCB 系列机械臂的 URDF 模型。

### 5. ./utils

功能工具目录，主要包含以下内容：

- 运动学库  
  - 正运动学 / 逆运动学求解  
  - 雅可比矩阵计算  
  - 直线轨迹规划  

- 基于 SDK 的二次开发功能  
  - 对 SDK 常用功能的进一步封装  
  - 可作为应用层开发参考

使用 utils 目录下的功能模块前，需要先安装依赖库：

```bash
pip install -r requirements.txt
```
